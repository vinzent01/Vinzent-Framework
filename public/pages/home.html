Features

  • Lightweight and fast.
  • Seamlessly integrates with Express.js.
  • Built-in support for routing and middleware.
  • Optimized for SEO and performance.
  • Deploy compatible with Vercel

How It Works

Vinzent Framework leverages the power of Express.js to serve pre-rendered HTML pages. With support for templates, you can dynamically serve static files

  1. create your site template inside public/template.html
  2. create pages inside public/pages
  3. load page components with jquery

Get Started

Follow these steps to get started:


# Install the framework
git clone https://github.com/vinzent01/Vinzent-Framework

# Build and serve
cd Vinzent-Framework

#install package dependencies
npm install

# to develop
npm run dev 

# to run
npm run start
